Cloud Provisioning - the processes for the deployment and integration of cloud computing services within an enterprise. It is a broad term that incorporates the policies, procedures, and an enterprise's objective in sourcing cloud services and solutions from a cloud service provider.

It primarily defines how, what and when an organization will provision cloud services. These services can be internal, public or hybrid cloud products and solutions.
